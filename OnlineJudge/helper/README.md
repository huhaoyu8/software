# 此仓库是关于OJ代码结构的注释、详解

# 所需知识

```sh
| Linux中的管道符号，左边命令的输出作为右边命令的输入
Nginx 配置中的root，	设定请求URL的本地文件根目录
systemctl enable xxx 当系统启动时，自动启动xxx服务
rc.local 开机启动文件
Crontab 一个用于管理、编辑和查询用户定时任务的命令行工具；https://www.runoob.com/linux/linux-comm-crontab.html
grep  标准输入中查找字符串 echo "hello world" | grep -c world
grep  在文件中查找文本 grep helloworld file.txt
screen 是Linux终端命令神器
ttr 转义命令 tr a-z A-Z
-gt 整型变量大于运算符 GreateThan
TTL 指定IP包被路由器丢弃前允许通过的最大网段数量Time To Live
echo $TERM 查看终端 $TERM是预定义标识符
awk 格式化读取命令
sed 编辑文件 a新增 c取代 d删除 i插入 p打印 s取代
sed 替换分隔符
sed -i "s#hello;#world;#g"s代表替换，g代表全局替换，'#'在这里用作分隔符，代替了通常使用的 /，这意味着可以用除了 / 以外的任何字符作为分隔符
chown 更改所有者(CHange OWNer) chown -R处理指定目录以及其子目录下的所有文件
clear 一键清理屏幕
start 自动搜索并打开指定的可执行文件
reset 重新初始化终端
client_max_body_size nginx的请求体的大小设置
MariaDB 是MySQL的一个分支，性能优于MySQL
mips 芯片每秒钟能够处理的指令数量
PHP-FPM（PHP FastCGI Process Manager）是PHP FastCGI管理器。它主要用于管理PHP FastCGI进程池，负责监听来自Web服务器的请求，并动态地管理PHP FastCGI子进程。PHP-FPM的出现是为了解决PHP FastCGI进程管理问题，能够平滑地管理多个进程池，并且具有更好的进程管理能力。
```
